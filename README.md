# JamberTech Pairing

WhatsApp Session ID Generator for KHAN-MD Bot.

## Deploy on Railway
1. Fork this repo
2. Go to Railway → New Project → Deploy from GitHub
3. Connect your fork — Railway auto-detects everything
4. Generate a domain in Settings → Networking

## Deploy on Render
1. Fork this repo
2. New Web Service → Connect GitHub
3. Build Command: `npm install`
4. Start Command: `node server.js`

## Features
- QR Code + Pair Code methods
- Session ID format: `DJ~<base64>`
- Dark WhatsApp-themed UI
- Auto session cleanup
